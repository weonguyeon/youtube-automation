## 2. 숏폼 상세 규격

### S8 (8초) — 극초단 포맷

**목적**: 스크롤 멈추기(Scroll-stopper). 한 가지 팩트/비교/질문만 전달.

```
구조:
[0~2초] Hook     — 충격적 숫자/질문/이미지
[2~6초] Content  — 핵심 정보 전달 (1~2컷)
[6~8초] CTA      — 구독/좋아요 유도 or 반전 한 줄

씬 수: 2~3개
자막: 줄당 8자 이내, 최대 1줄
내레이션: 없음 or 한 문장 (선택)
BGM: 트렌디/임팩트 (8초 클립)
```

**대본 JSON 구조:**
```json
{
  "format": "S8",
  "duration_sec": 8,
  "scenes": [
    {
      "scene_id": 1,
      "type": "hook",
      "duration_sec": 2,
      "subtitle": "지구에서 가장 비싼 물질",
      "visual_prompt": "..."
    },
    {
      "scene_id": 2,
      "type": "content",
      "duration_sec": 4,
      "subtitle": "1g에 620억 원",
      "visual_prompt": "..."
    },
    {
      "scene_id": 3,
      "type": "cta",
      "duration_sec": 2,
      "subtitle": "그게 뭘까?",
      "visual_prompt": "..."
    }
  ]
}
```

**패턴별 S8 적용:**
| 패턴 | S8 활용 예시 |
|------|-------------|
| A (데이터) | 8초 게이지 바 채워지기, 숫자 카운트업 |
| B (텍스트) | 한 줄 팩트 + 큰 자막 + 배경 이미지 |
| C (무음) | 인포그래픽 카드 1장 + 핵심 숫자 |

### S15 (15초) — 미니 인포그래픽

**목적**: 간결한 스토리. 비교/순위/프로세스를 빠르게 전달.

```
구조:
[0~3초]  Hook      — 질문/충격 통계
[3~12초] Content   — 핵심 정보 3~4컷 (컷당 2~3초)
[12~15초] Wrap-up  — 결론 한 줄 + CTA

씬 수: 4~6개
자막: 줄당 12자 이내, 최대 2줄
내레이션: TTS 사용 (또는 무음)
BGM: 트렌디/정보성 (15초 루프)
```

**대본 JSON 구조:**
```json

**패턴별 S15 적용:**

|------|-------------|

| B (텍스트) | 5컷 팩트 전달, 비교 인포그래픽 |
| C (무음) | 인포그래픽 카드 3~4장 슬라이드 |

## 3. 롱폼 상세 규격 (이후 개발)

### L3 (3분)
```
구조:
[0~15초]   Hook + 주제 소개
[15초~2분] 본론 (3~4개 포인트)
[2~3분]    결론 + CTA

씬 수: 12~15개
자막: 줄당 20자, 최대 2줄 (하단)
```

### L5 (5분)
```
구조:
[0~30초]   Hook + 배경 설명
[30초~4분] 본론 (5~6개 포인트, 각 40초)
[4~5분]    결론 + 인사이트 + CTA

씬 수: 20~25개
```

### L10 (10분)
```
구조:
[0~1분]    Hook + 큰 질문 제기
[1~3분]    배경/맥락 설명
[3~8분]    본론 (7~10개 포인트)
[8~9분]    반전/인사이트
[9~10분]   결론 + CTA

씬 수: 35~45개
```

## 4. 공통 규격

### 자막 스타일 (포맷별)
| 규격 | S8 | S15 | L3/L5/L10 |
|------|-----|------|-----------|
| 위치 | 중앙 (50%) | 중앙 (60%) | 하단 (85%) |
| 폰트 크기 | 72px | 60px | 42px |
| 최대 줄 수 | 1 | 2 | 2 |
| 줄당 글자 | 8자 | 12자 | 20자 |
| 강조 색상 | accent color | accent color | accent color |

### 트랜지션 (포맷별)
| 규격 | S8 | S15 | L3/L5/L10 |
|------|-----|------|-----------|
| 기본 전환 | hard cut | cut / zoom | fade / slide |
| 전환 속도 | 0ms | 200ms | 300~500ms |
| 줌 효과 | 강함 (1.2x) | 중간 (1.1x) | 약함 (1.05x) |

### BGM 볼륨
| 규격 | 내레이션 있을 때 | 내레이션 없을 때 |
|------|----------------|----------------|
| S8 | 0.3 | 0.7 |
| S15 | 0.15 | 0.6 |
| L3/L5/L10 | 0.1 | 0.5 |

## 5. 개발 우선순위

```

  ├── S8  × Pattern A (데이터) ← MVP 최초 구현
  ├── S8  × Pattern B (텍스트)
  ├── S8  × Pattern C (무음)
  ├── S15 × Pattern A (데이터)
  ├── S15 × Pattern B (텍스트)
  └── S15 × Pattern C (무음)

Phase 2 (롱폼)

Phase 3 (고급)
  └── 모든 포맷 × Pattern F (3D)
